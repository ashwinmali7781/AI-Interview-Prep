# AI Interview Prep

A full-stack web application to master coding interviews — built with React, Vite, Supabase, and Claude AI.

## Features
- AI Code Review (Claude API)
- Coding Practice with Monaco Editor
- AI Mock Interviews
- Daily Streak Tracker
- Leaderboard
- Dark Mode
- Company Filters

## Setup
```bash
npm install
# add .env with VITE_SUPABASE_URL and VITE_SUPABASE_PUBLISHABLE_KEY
npm run dev
```

## Deploy
Build command: `npm install && npm run build`
Publish directory: `dist`
